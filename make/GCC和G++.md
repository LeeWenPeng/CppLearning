## 1  GCC 和 G++

GCC 和 G++ 都是 GUN 组织的编译器，二者功能几乎类似，但仍有部分不同：

|          | GCC        | G++        |
| :------- | ---------- | :--------- |
| `.c`文件 | C 程序    | C++程序 |
| `.cpp`文件 | C++程序 | C++程序 |
| 编译     | 能         | 自动调用 `gcc` |
| 链接 C库 | 能 | 能 |
| 链接 C++ 库 | `gcc -lstdc++` | 能 |
| `__cplusplus` | 未定义 | 已定义 |

GCC（GNU Compiler Collection）提供了丰富的编译选项，用于控制编译过程、优化代码、调试等。
