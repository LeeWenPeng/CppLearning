## 1  枚举类型的定义

```cpp
// 定义 1
enum enum_type_name
{
	ENUM_VALUE_1,
	ENUM_VALUE_2,
	ENUM_VALUE_3,
	...
};
```

- enum_type_name：用户定义的枚举类型名
- ENUM_VALUE_N：枚举值
- 一般枚举值为全大写字母  

## 2  默认值

默认情况下，枚举值从 0 开始，每次增加 1。

```cpp
// 定义 2
enum enum_type_name
{
	ENUM_VALUE_1, // 值为 0 
	ENUM_VALUE_2, // 值为 1
	ENUM_VALUE_3, // 值为 2
	…
};
```

## 3  定义时为枚举值指定特定的值

枚举值可以在定义时指定特定的值，对于没有指定值的枚举值，其值为其前一个枚举值加 1。

```cpp
enum enum_type_name
{
	ENUM_VALUE_1 = 50, // 值为 50 
	ENUM_VALUE_2, // 值为 51
	ENUM_VALUE_3 = 100, // 值为 100
	…
};
```

## 4  声明一个枚举类型的变量

```cpp
enum_type_name variable = ENUM_VALUE_N;
```

## 5  枚举值可以进行算术运算

```cpp
cout << ENUM_VALUE_1 + ENUM_VALUE_3 << endl; // 150
```
